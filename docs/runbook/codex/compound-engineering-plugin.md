# Compound Engineering plugin

## Purpose

The Compound Engineering plugin adds CE workflow skills to Codex, including planning, review, debug, browser QA, iOS test, commit, PR, and documentation flows.

## Install

```sh
codex plugin marketplace add EveryInc/compound-engineering-plugin
codex plugin add compound-engineering@compound-engineering-plugin
```

Restart Codex after install so the new skills load.
